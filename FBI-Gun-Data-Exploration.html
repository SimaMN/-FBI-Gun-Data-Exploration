Introduction

Dataset Description

NICS Data (from the FBI's National Instant Criminal Background Check System):

This dataset contains information on firearm checks conducted across different states in the U.S. It is structured by month and state, detailing the types of checks conducted and their counts.

Columns:

month: The month of the firearm check (e.g., "2017-09").

state: The state where the firearm check occurred.

permit: The number of permit checks.

permit_recheck: Number of rechecks on permits.

handgun, long_gun, other: Number of checks for handguns, long guns, and other types of firearms, respectively.

multiple: Checks involving multiple firearm types.

admin: Administrative checks.

prepawn_handgun, prepawn_long_gun, prepawn_other: Checks related to pawning firearms.

redemption_handgun, redemption_long_gun, redemption_other: Redemption checks for previously pawned firearms.

returned_handgun, returned_long_gun, returned_other: Checks for returned firearms.

rentals_handgun, rentals_long_gun: Checks related to rentals.

private_sale_handgun, private_sale_long_gun, private_sale_other: Checks for private sales. return_to_seller_handgun, return_to_seller_long_gun, return_to_seller_other: Checks for firearms returned to sellers.

totals: The total number of checks conducted in a given month for a state.

U.S. Census Data (from census.gov):

This dataset contains demographic and economic data for each state in the U.S., primarily for the year 2016.

Columns:

Fact: A description of the variable (e.g., population size, income levels).

Fact Note: Additional notes about the variable.

The remaining columns represent each state (e.g., Alabama, Alaska, etc.) with data values corresponding to the variable described in the Fact column.

Relationship Between Tables

The NICS Data table provides monthly information about firearm background checks for each state, while the Census Data provides demographic and socioeconomic information for each state. These tables can be related using the state attribute in the NICS dataset and the state columns in the Census dataset, enabling state-level analysis of firearm checks in relation to demographic factors.

This question examines the overall trend of gun purchases across the United States over time. By analyzing the data, the goal is to identify patterns such as seasonality, spikes in purchases, or overall growth or decline in firearm acquisitions.

These questions will guide the analysis to provide a comprehensive view of firearm registration trends in relation to demographic and economic data. The answers to these questions will help illustrate the relationship between socioeconomic factors and firearm ownership patterns.

Data Cleaning

Data Structure Overview

NICS Dataset:

This data contains 27 columns, including: month (string): The month and year of firearm checks. state (string): The state where the checks were conducted. Various columns for different types of firearm checks (e.g., handgun, long_gun, permit). Aggregate columns like totals showing the total checks per month per state.

Potential Problems:

Missing values in several columns (e.g., permit_recheck, other, returned_*). Inconsistent data types (e.g., some numeric columns might have been read as strings if they contained errors). Duplicate rows or columns with little variation.

Census Dataset:

This datancontains 52 columns, including: Fact (string): Descriptions of different demographic and economic factors. Fact Note (string): Notes providing additional information on the Fact. Columns for each state containing values related to the demographic/economic factor.

Potential Problems:

Missing values in Fact and Fact Note. Data in state columns may contain inconsistent formats (e.g., percentages stored as strings). Some entries may not be relevant to the analysis (e.g., demographic data unrelated to firearm ownership).

Data Cleaning Steps

NICS Dataset:

  • Drop columns with excessive missing values: Columns like permit_recheck, other, and returned_* may not provide sufficient information due to high missing values.

  • Fill missing values in remaining columns: Use median values for numeric columns to fill remaining gaps.

  • Convert data types: Ensure all numeric columns are appropriately set as float or int types.

  • Remove duplicate rows: Check for and remove any duplicate entries.

Census Dataset:

  • Drop rows with missing Fact values: Rows missing key identifiers (Fact) will be removed as they may not be informative.

  • Fill missing values in Fact Note: Fill missing notes with empty strings as they are supplementary.

  • Convert percentages to numeric: Clean state columns by converting percentage strings to numeric values (e.g., removing the % sign and dividing by 100).

  • Filter relevant rows: Retain rows with demographic and economic factors most likely relevant to firearm ownership (e.g., population size, median income).

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Step 1: Load and preprocess the NICS data from the specified directory
nics_data = pd.read_csv('.../project_investigate a dataset/gun_data.csv')

# Convert 'month' to datetime and extract 'year'
nics_data['month'] = pd.to_datetime(nics_data['month'], errors='coerce')
nics_data = nics_data.dropna(subset=['month'])  # Drop invalid rows
nics_data['year'] = nics_data['month'].dt.year

# Retain necessary columns and drop rows with zero or missing totals
nics_data_cleaned = nics_data[['state', 'year', 'totals']].dropna()
nics_data_cleaned = nics_data_cleaned[nics_data_cleaned['totals'] > 0]

# Step 2: Group by state and year, then calculate the annual totals
state_yearly_totals = nics_data_cleaned.groupby(['state', 'year'])['totals'].sum().reset_index()

# Step 3: Calculate the year-over-year percentage growth
state_yearly_totals['growth'] = state_yearly_totals.groupby('state')['totals'].pct_change() * 100

# Filter out rows with missing or infinite growth values
state_yearly_totals = state_yearly_totals.dropna(subset=['growth'])
state_yearly_totals = state_yearly_totals[~np.isinf(state_yearly_totals['growth'])]

# Filter out states with no variation in totals
variation_by_state = state_yearly_totals.groupby('state')['totals'].std()
valid_states = variation_by_state[variation_by_state > 0].index  # Keep states with variation
state_yearly_totals = state_yearly_totals[state_yearly_totals['state'].isin(valid_states)]

# Step 4: Exclude territories and calculate the average growth by state
excluded_territories = ['Virgin Islands', 'Mariana Islands', 'Guam', 'Puerto Rico', 'American Samoa']
state_yearly_totals = state_yearly_totals[~state_yearly_totals['state'].isin(excluded_territories)]

# Calculate the average growth for states only
average_growth_by_state = state_yearly_totals.groupby('state')['growth'].mean().reset_index()
average_growth_by_state = average_growth_by_state.sort_values(by='growth', ascending=False)

# Display the top 5 states with the highest growth in gun registrations
top_5_states = average_growth_by_state.head(5)

# Visualization: Trends for the top 5 states with formatted x-axis ticks
plt.figure(figsize=(12, 8))
top_states = top_5_states['state'].tolist()

for state in top_states:
    state_data = state_yearly_totals[state_yearly_totals['state'] == state]
    plt.plot(state_data['year'], state_data['totals'], marker='o', label=state)

# Add labels, title, and legend
plt.title('Trends in Gun Registrations for Top 5 States with Highest Growth')
plt.xlabel('Year')
plt.ylabel('Total Gun Registrations')
plt.legend(title='State')
plt.grid(True, linestyle='--', alpha=0.7)

# Format x-axis ticks as integers
plt.xticks(ticks=state_yearly_totals['year'].unique(), labels=state_yearly_totals['year'].unique().astype(int))

Conclusions

The findings indicate that economic factors such as median household income significantly correlate with gun ownership rates, while certain states have experienced exceptionally high growth in gun registrations. Additionally, the overall trend suggests an increasing pattern in gun purchases, influenced by various external and societal factors over time. These insights provide a comprehensive view of the factors associated with gun ownership and the patterns in gun registration growth across the United States.

One limitation of this analysis is that it relies on the assumption that firearm background checks directly correlate with gun purchases. However, background checks may be conducted for other reasons (e.g., permit renewals), and not all background checks result in a gun purchase. This could potentially affect the accuracy of using background check data as a proxy for actual gun ownership rates and growth patterns.
